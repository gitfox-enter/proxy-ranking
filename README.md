# Proxy Tools Ranking

A curated, auto-updated ranking of the top 30 proxy-related open-source projects on GitHub, covering clients, subscription services, and management tools for Windows and Android.

## Ranking Coverage

- **Proxy clients** — GUI and CLI clients for Windows, Android, and cross-platform (Clash Verge, v2rayN, v2rayNG, sing-box, etc.)
- **Free subscriptions** — Community-maintained free node subscriptions and aggregators
- **Management tools** — Subscription converters, web panels, and one-click deployment scripts

## How It Works

The ranking is compiled periodically via GitHub Actions using the GitHub Search API, filtered for proxy-related keywords, sorted by star count, and deployed to GitHub Pages as a static site.
